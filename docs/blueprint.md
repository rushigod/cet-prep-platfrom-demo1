# **App Name**: CET Prep Platform

## Core Features:

- Exam Simulation: Simulate the MHT CET exam environment with a timer and question format that mirrors the actual CBT.
- Test Scheduling: Allow administrators to schedule tests with specific start and end times (e.g., 12 PM to 3 PM), providing a fixed testing window for students.
- Real-time Timer: Implement a real-time timer that accurately reflects the remaining time in the test, irrespective of when a student starts within the allocated window. Students will see the actual remaining time from the exam's designated start time.
- Sectioned Tests: Divide tests into Physics & Chemistry and Mathematics sections, matching the MHT CET structure.
- Manual Question Import: Text area supporting the manual input or pasting of test questions.
- CSV Upload: Enable uploading test questions in CSV format for bulk import.
- Result Storage: Persistently save test results for each student.

## Style Guidelines:

- Primary color: Deep Blue (#293462) to inspire focus and confidence.
- Background color: Light blue (#D6E4FF) for a calm, neutral backdrop.
- Accent color: Vivid Purple (#A663CC) to highlight important elements like timers and calls to action.
- Body font: 'Inter' sans-serif for clear readability of test questions and instructions.
- Headline font: 'Space Grotesk' sans-serif, used for headers
- Use clear, understandable icons for navigation and test controls.
- Subtle animations to provide feedback and enhance user experience.
